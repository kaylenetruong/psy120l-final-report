The Role of Attachment Style in Perceived Compatibility and Social Attraction in Romantic Relationships : Effects of High vs. Low Communication in Dating Profiles
This repository contains the published HTML version of myPSY 120L final reproducible report.
This project examined how familial communication patterns influence romantic partner preferences, and if attachment security moderates this relationship.
** The report was created using Quarto and R.
** The published report file is index.html.
** The dataset is not included in this repository
